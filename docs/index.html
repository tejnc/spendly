Split Tracker Cloud

Splitwise-style group expense tracking with real-time sync — privacy-isolated from your local data. Create groups, invite members with a simple code, and settle balances without awkward conversations.

  • Create groups, invite members instantly

  • Live balance summaries & settlement tracking
  • Group activity feed with full history
  • Multi-payer support per expense
  • Real-time sync across all group members
  • Isolated cloud module — personal data stays local

Privacy First

Your core data never leaves your device

Spendly requires no account, no sign-in, and no internet connection for core features. Every transaction, account, and budget is stored exclusively in your phone's local database using Isar — fast, encrypted, and 100% private. The optional cloud Split Tracker is fully isolated and only activates when you choose to enable it.

No cloud sync No account required No data mining Offline first Biometric lock Encrypted backup
