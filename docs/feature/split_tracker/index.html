How It Works

🔒

Privacy-Isolated By Design

Split Tracker is a self-contained cloud module. Your personal transactions, budgets, accounts, and categories are stored exclusively on your device and are completely untouched by the cloud layer — even when Split Tracker is active.

  • Lazy initialization — no network connection opened until you deliberately enable Split Tracker
  • Cloud module lives in a separate package with no access to local Isar data
  • Disabling Split Tracker closes all network connections immediately
  • Personal finance data never leaves your device
  • No cross-contamination between local and cloud data
  • Opt-in, not opt-out — cloud only activates on your terms
🛡️
Privacy First

Your personal data never touches the cloud

Split Tracker uses Supabase for real-time sync, but only for group-shared data you explicitly add to a group. Your transactions, budgets, accounts, and personal finances remain 100% local. The cloud module is lazily initialized — no network connection is opened until you actively launch Split Tracker for the first time.
